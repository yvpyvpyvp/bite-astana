# BITE

AI restaurant finder for Astana, Kazakhstan.

Mobile-first React + Tailwind CSS MVP with mock data and API-ready architecture.

## Stack
- React
- Vite
- Tailwind CSS
- Mock data for Astana restaurants

## Env
Create `.env` with:

```bash
TWO_GIS_API_KEY=your_key_here
AI_API_KEY=your_key_here
```

## Run
```bash
npm install
npm run dev
```
